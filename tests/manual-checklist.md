# 수동 테스트 체크리스트 (브라우저에서 직접 확인)

## 1. 메인 페이지 (index.html)
### 정상 흐름
- [ ] 브라우저에서 https://무료부고장.com 접속 -> 기대: 랜딩 페이지 로딩, 파비콘 표시
- [ ] "부고장 작성하기" 버튼 클릭 -> 기대: write.html로 이동
- [ ] 템플릿 샘플 이미지 33개가 그리드로 표시되는지 확인
- [ ] 사이드바 메뉴 버튼 클릭 -> 기대: 사이드바 슬라이드, 모든 메뉴 링크 표시
- [ ] FAQ 섹션 질문 클릭 -> 기대: 답변 아코디언 펼쳐짐
### 모바일 (375px)
- [ ] 화면 375px에서 레이아웃 깨짐 없는지 확인
- [ ] 사이드바가 전체 너비로 열리는지 확인

## 2. 부고장 작성 (write.html)
### 정상 흐름 (전체 작성)
- [ ] 템플릿 선택: 상단 템플릿 이미지 중 1개 클릭 -> 기대: 선택된 템플릿 파란 테두리 표시
- [ ] 고인 성함: "홍길동" 입력 -> 기대: 글자 입력됨, 50자 제한
- [ ] 향년: "80" 입력 -> 기대: 숫자만 입력 가능
- [ ] 성별: "남성" 선택 -> 기대: 드롭다운에서 선택
- [ ] 별세표현: "별세" 선택
- [ ] 별세일: 날짜 선택기에서 날짜 선택 -> 기대: 날짜 표시
- [ ] 별세시간: 시간 선택기에서 시간 선택 -> 기대: HH:MM 표시
- [ ] 장례식장 검색: "서울" 입력 후 검색 -> 기대: 결과 목록 표시
- [ ] 결과에서 장례식장 선택 -> 기대: 이름, 주소, 전화번호 자동 입력
- [ ] 빈소: "1층 101호" 입력
- [ ] 발인일/시간: 날짜와 시간 선택
- [ ] 유가족 관계: "장남" 선택, 이름: "홍길순" 입력
- [ ] 유가족 추가 버튼 클릭 -> 기대: 새 행 추가
- [ ] 비밀번호: "1234" 입력 (4자리 숫자)
- [ ] 약관 동의 체크박스 클릭
- [ ] "부고장 만들기" 버튼 클릭 -> 기대: 저장 성공, preview.html로 이동
### 예외 상황
- [ ] 고인 성함 비우고 제출 -> 기대: "고인 성함을 입력해주세요" 토스트 + 해당 필드로 스크롤
- [ ] 비밀번호 3자리만 입력 후 제출 -> 기대: 비밀번호 에러 메시지
- [ ] 템플릿 선택 안 하고 제출 -> 기대: 템플릿 선택 에러
- [ ] 유가족 이름 비우고 제출 -> 기대: 유가족 정보 에러

## 3. 부고장 미리보기 (preview.html)
### 정상 흐름
- [ ] write.html에서 저장 후 자동 이동 -> 기대: 템플릿+정보 표시
- [ ] 고인명, 향년, 별세일시, 장례식장, 발인일시, 유가족 정보 모두 표시 확인
- [ ] "OO명이 조문하셨습니다" 조회수 표시 확인
- [ ] 새로고침 -> 기대: 조회수 1 증가
- [ ] "공유" 버튼 클릭 -> 기대: 공유 모달 (카카오톡, 문자, 링크, 밴드, QR) 표시
- [ ] "링크" 클릭 -> 기대: 클립보드에 URL 복사, 토스트 "복사되었습니다"
- [ ] "QR코드" 클릭 -> 기대: QR코드 이미지 모달 표시
- [ ] 지도보기 버튼 클릭 -> 기대: 네이버 지도 새 탭으로 열림
- [ ] 주소 복사 버튼 클릭 -> 기대: 장례식장 주소 복사됨
### 방명록 (조문 메시지)
- [ ] 이름: "김조문", 관계: "친구", 메시지: "명복을 빕니다" 입력 후 등록 버튼 클릭 -> 기대: 방명록에 추가, 목록 표시
- [ ] 이름 비우고 등록 -> 기대: 에러 토스트
- [ ] 메시지 비우고 등록 -> 기대: 에러 토스트
- [ ] 방명록 삭제 버튼 클릭 -> 기대: 비밀번호 입력 요청
- [ ] 올바른 비밀번호 "1234" 입력 -> 기대: 삭제 성공
### 인쇄
- [ ] "인쇄" 버튼 클릭 -> 기대: 인쇄 대화상자, UI 요소 숨김, 본문만 표시
### 모바일 (375px)
- [ ] 모든 섹션 세로 스크롤로 확인 가능
- [ ] 공유 모달이 화면 너비에 맞게 표시

## 4. 부고장 검색 (Search.html)
### 정상 흐름
- [ ] 고인명: "홍길동", 비밀번호: "1234" 입력 후 검색 버튼 클릭 -> 기대: 검색 결과 표시
- [ ] 결과에서 "부고장 보기" 클릭 -> 기대: preview.html로 이동
- [ ] 결과에서 "부고장 관리" 클릭 -> 기대: manage.html로 이동
### 예외 상황
- [ ] 고인명만 입력하고 검색 -> 기대: "비밀번호를 입력해주세요" 토스트
- [ ] 존재하지 않는 이름 검색 -> 기대: "부고장을 찾을 수 없습니다" 토스트
- [ ] 틀린 비밀번호로 검색 -> 기대: "비밀번호가 일치하지 않습니다" 토스트

## 5. 부고장 관리 (manage.html)
### 정상 흐름
- [ ] Search에서 이동 후 -> 기대: 통계 카드 (조회수, 조문 메시지, 근조화환, 만료일) 표시
- [ ] 생성일/수정일 표시 확인
- [ ] "수정" 버튼 클릭 -> 기대: 비밀번호 입력 모달
- [ ] 비밀번호 "1234" 입력 -> 기대: write.html 수정 모드로 이동
- [ ] 수정 모드에서 고인명 변경 후 저장 -> 기대: 변경사항 반영
- [ ] "삭제" 버튼 클릭 -> 기대: 확인 모달 표시
- [ ] 확인 후 비밀번호 입력 -> 기대: 삭제 완료, 메인으로 이동
### 예외 상황
- [ ] 틀린 비밀번호로 수정 시도 -> 기대: "비밀번호가 일치하지 않습니다"
- [ ] URL에 잘못된 ID 직접 입력 -> 기대: 에러 메시지 표시

## 6. 정적 페이지
### guide.html (이용안내)
- [ ] 페이지 접속 -> 기대: 모든 기능 안내 표시 (QR, 밴드, 방명록, 인쇄 등)
### faq.html (FAQ)
- [ ] 각 질문 클릭 -> 기대: 아코디언 답변 펼쳐짐/접힘
- [ ] 보관 기간 답변 -> 기대: "발인일로부터 7일간 공개, 30일 자동 삭제"
### etiquette.html (조문예절)
- [ ] 6개 아코디언 섹션 모두 클릭하여 펼쳐짐 확인
### procedure.html (장례절차)
- [ ] 타임라인, 체크리스트, 비용 가이드, 지원제도 섹션 표시 확인
### sample.html (템플릿 샘플)
- [ ] 샘플 부고장 표시 확인
### thanks.html (감사장)
- [ ] 폼 입력 후 복사 버튼 -> 기대: 감사장 텍스트 클립보드 복사
### 404.html (에러 페이지)
- [ ] 존재하지 않는 URL 접속 -> 기대: 404 페이지, 홈/검색/작성 링크 표시

## 7. 키보드 접근성 (WCAG)
- [ ] Tab키로 모든 버튼/링크 순차 포커스 이동 확인
- [ ] 템플릿 선택: Tab으로 이동 후 Enter/Space -> 기대: 템플릿 선택됨
- [ ] 공유 모달 버튼: Tab으로 이동 후 Enter -> 기대: 공유 동작
- [ ] 모달 열린 상태에서 Escape -> 기대: 모달 닫힘
- [ ] 모달 닫힌 후 포커스가 트리거 버튼으로 복귀
